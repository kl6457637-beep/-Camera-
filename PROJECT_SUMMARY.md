## 🚀 快速开始

### 方式1：使用网页预览（推荐先看效果）

1. 打开浏览器访问：`http://localhost:8080/preview.html`
2. 或者双击打开 `preview.html` 文件

### 方式2：启动微信小程序开发

1. 双击运行 `start-dev.bat`
2. 等待依赖安装完成
3. 使用微信开发者工具导入项目

### 方式3：命令行启动

```bash
cd photo-portfolio
npm install --legacy-peer-deps
npm run dev:weapp
```

## 🆘 常见问题

### Q: 如何修改摄影师信息？
A: 修改 `src/app.tsx` 中的模拟数据，或连接云开发数据库

### Q: 如何添加自己的作品？
A: 在管理页上传，或在云开发数据库 → photos 集合中添加

### Q: 云开发收费吗？
A: 有免费额度：5GB存储、5GB数据库、50万次调用/月

### Q: 可以部署到其他平台吗？
A: 可以！Taro支持：微信、支付宝、百度、字节跳动、QQ、H5
