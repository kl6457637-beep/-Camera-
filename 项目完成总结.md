## 🚀 快速启动指南

### 1. 环境准备
```bash
# 1. 安装 Node.js (版本 >= 14)
# 2. 安装微信开发者工具

# 3. 安装项目依赖
cd C:\Users\admin\photo-portfolio
npm install
```

### 2. 配置云开发
```typescript
// 编辑 src/app.tsx
Taro.cloud.init({
  env: 'your-cloud-env-id'  // 替换为你的云环境ID
})
```

### 3. 启动开发
```bash
# 微信小程序开发
npm run dev:weapp

# H5预览
npm run dev:h5
```

### 4. 云函数部署
1. 开通微信云开发
2. 创建环境
3. 右键 cloudfunctions 文件夹 → 上传并部署

### 5. 数据库配置
创建以下集合：
- works（作品）
- bookings（预约）
- favorites（收藏）
- photographers（摄影师信息）
