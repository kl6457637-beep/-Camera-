# 📷 摄影师个人作品集小程序

> 一个专为摄影师设计的作品集小程序，采用小红书风格设计，具备强大的意向收集功能。

## ✨ 核心特性

### 🎨 视觉设计
- **小红书风格**：温暖活泼的粉橙色调，圆角卡片设计
- **瀑布流布局**：非对称排版，增加视觉灵动感
- **沉浸式浏览**：全屏大图展示，自动隐藏UI元素
- **精致动效**：图片懒加载模糊渐变、心跳动画、滑入效果

### 💝 意向收集（核心功能）
- **心动收藏**：模特点击爱心即可收藏喜欢的作品
- **风格分析**：自动统计用户偏好的风格类型
- **意向同步**：约拍时自动将收藏列表同步给摄影师
- **专业开场白**：摄影师可以看到用户具体心动了哪些作品

### 📱 页面功能

#### 访客端
- **首页**：摄影师信息展示 + 瀑布流作品列表
- **详情页**：沉浸式大图浏览 + 心动/约拍操作
- **收藏页**：心动清单展示 + 风格分组统计
- **联系页**：意向卡片 + 约拍流程说明 + 微信复制

#### 管理端
- **作品上传**：批量选择图片 + 风格标签分类
- **意向查看**：实时查看模特意向反馈 + 联系状态管理

## 🚀 快速开始

### 1. 安装依赖

```bash
cd photo-portfolio
npm install --legacy-peer-deps
```

### 2. 配置云开发

1. 登录[微信公众平台](https://mp.weixin.qq.com/)开通云开发
2. 获取环境ID
3. 修改 `src/app.tsx` 中的 `env` 配置
4. 按照 `CLOUD_DEVELOPMENT_GUIDE.md` 创建数据库和云函数

### 3. 运行项目

```bash
# 开发模式
npm run dev:weapp

# 构建
npm run build:weapp
```

## ☁️ 云开发架构

### 数据库集合
- **photos**: 作品数据
- **styles**: 风格分类
- **intents**: 用户意向
- **photographer**: 摄影师信息

### 云函数
- **submitIntent**: 提交用户意向
- **getIntents**: 获取意向列表
- **updateIntentStatus**: 更新意向状态
- **uploadPhoto**: 上传作品

## 💡 使用场景

### 场景1：模特浏览作品
1. 进入小程序，浏览瀑布流作品
2. 点击喜欢的作品进入详情页
3. 点击「心动」收藏作品
4. 查看多个作品后，进入心动清单

### 场景2：模特发起约拍
1. 在心动清单页点击「联系摄影师」
2. 查看意向卡片（展示收藏统计）
3. 阅读约拍流程说明
4. 点击「确认意向并联系摄影师」
5. 复制微信号，发送定制开场白

### 场景3：摄影师查看意向
1. 进入管理页，切换到「意向反馈」
2. 查看待联系的意向列表
3. 看到用户心动了哪些风格和作品
4. 标记为已联系，准备沟通

## 🔒 数据安全

- 意向数据通过微信 OpenID 关联
