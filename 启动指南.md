# 🚀 快速启动指南

## 方式1: 使用启动脚本（推荐）

双击运行：
```
启动开发环境.bat
```

## 方式2: 手动启动

```bash
# 1. 进入项目目录
cd C:\Users\admin\photo-portfolio

# 2. 安装依赖
npm install --legacy-peer-deps

# 3. 启动H5开发服务器
npm run dev:h5
```

## 访问地址

- **H5预览**: http://localhost:10086
- **微信小程序**: 使用微信开发者工具打开项目目录

## 常用命令

| 命令 | 说明 |
|------|------|
| `npm run dev:h5` | 启动H5开发服务器 |
| `npm run build:h5` | 构建H5版本 |
| `npm run dev:weapp` | 启动微信小程序开发 |
| `npm run build:weapp` | 构建微信小程序 |

## 注意事项

1. 确保Node.js版本 >= 18
2. 首次运行需要安装依赖，可能需要几分钟
3. H5版本可以直接在浏览器预览
4. 小程序版本需要微信开发者工具
5. 如遇依赖问题，运行 `npm install --legacy-peer-deps`

## 常见问题

### Q: 依赖安装失败
A: 运行 `npm install --legacy-peer-deps` 尝试

### Q: 端口被占用
A: 修改 `config/dev.js` 中的端口配置

### Q: 编译错误
A: 检查Node.js版本，确保 >= 18
